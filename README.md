# 🎨 Colour Analysis App

A simple and interactive web application that helps users discover their ideal colour palette based on three key personal features: **undertone**, **contrast**, and **depth**.

---

## 🚀 Overview

The Colour Analysis App guides users through a short quiz and recommends a personalised colour palette that complements their natural appearance.

This project is inspired by personal styling techniques used in fashion and design, making it both practical and engaging.

---

## ✨ Features

* 🎯 3-question guided quiz:

  * Undertone (Warm / Cool / Neutral)
  * Contrast (Low / Medium / High)
  * Depth (Light / Medium / Deep)

* 🎨 Instant colour palette recommendation

* 📱 Responsive and user-friendly UI

* ⚡ Fast and lightweight

---

## 🧠 How It Works

The app uses a simple rule-based system:

1. **Undertone**

   * Warm → Earthy tones (yellow, orange, olive)
   * Cool → Jewel tones (blue, purple, pink)
   * Neutral → Balanced mix

2. **Contrast**

   * Low → Soft, blended colours
   * High → Bold, contrasting colours

3. **Depth**

   * Light → Pastels
   * Medium → Balanced tones
   * Deep → Rich, dark colours

Based on user inputs, the app maps to a predefined colour palette.

---
## 🧪 Example Flow

1. User selects:

   * Undertone: Warm
   * Contrast: High
   * Depth: Deep

2. App returns:

   * 🔥 Rich warm palette (mustard, rust, deep olive, burgundy)
